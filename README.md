# Homework 2 - Build and Evaluate a Simple GenAI Workflow

## Workflow chosen
Compliance Risk Translator

## User
A project coordinator, compliance lead, or operations manager

## Input
Unstructured compliance notes, governance observations, or risk-related meeting notes

## Output
A structured summary that includes:
- risk
- action items
- owner
- escalation level
- notes

## Why this task is valuable
Compliance and risk-related notes are often messy, ambiguous, and time-sensitive. A first-pass structured draft can help teams identify risks faster, improve communication, and support follow-up while still keeping a human in the loop for review.
